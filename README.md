# WanAndroidClient
Try to build a www.wanandroid.com client

## 接口
[玩Android接口](http://www.wanandroid.com/blog/2)

## 代码结构
**Kotlin + MVP + Kotlin-Coroutines + Retrofit2(GsonCallAdapterFactory + CoroutineCallAdapterFactory)**

## 功能
- [x] 首页
- [x] 知识体系
- [x] 知识体系专题
- [x] 搜索
- [x] 登录、注册、注销
- [x] 我的收藏、取消收藏、添加收藏
- [x] 文章内容
- [x] 分享文章、专题
- [x] 常用网站
- [x] 热词搜索
- [ ] 搜索记录
- [ ] 离线数据
